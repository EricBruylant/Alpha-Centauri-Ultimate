Feminine Union Faction

------------------------------------------------------------------------

HOW TO USE: 

ORIGINAL SMAC: Unzip then rename the files to the name of another faction(union3.pcx
becomes peace.pcx etc) then copy them into you SMAC directory, REMEMBER TO BACK UP 1st!


ALIEN CROSSFIRE: Unzip the files to your SMAC directory then run Facedit.exe (in same directory)
Then, in FACEDIT load union.txt, save it then click add faction to game. Close Facedit then run
Alien Crossfire.
